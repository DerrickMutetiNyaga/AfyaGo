# PWA Icons

This folder should contain the following icon files for the PWA:

- `icon-192.png` - 192x192 pixels
- `icon-512.png` - 512x512 pixels

## How to Generate Icons

You can use online tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://favicon.io/

Or use image editing software to create square PNG images at the required sizes.

The icons should:
- Be square (same width and height)
- Have a transparent or solid background
- Represent the Afyago brand/logo
- Be optimized for web use
